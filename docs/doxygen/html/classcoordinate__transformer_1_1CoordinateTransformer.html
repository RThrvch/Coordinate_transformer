Конструктор для юнит-тестирования с инъекцией зависимостей.

Инициализирует CoordinateTransformer, используя переданный узел ROS 2 и предоставленный буфер TF2. Не создает внутреннего слушателя TF2.

Exceptions
std::invalid_argumentесли node или tf_buffer равны nullptr.

Преобразует позу из исходной системы координат в целевую.

Использует буфер TF2 для выполнения преобразования. Проверяет, находится ли результирующая точка в пределах границ, установленных для target_frame.

Parameters
inputВходная поза (PoseStamped), frame_id которой указывает исходную систему координат.
outputВыходная поза (PoseStamped), в которую будет записан результат преобразования в системе target_frame.
target_frameИдентификатор целевой системы координат (TF frame ID).
Returns
Статус операции: SUCCESS, OUT_OF_BOUNDS, TRANSFORM_NOT_FOUND, INVALID_INPUT или CONFIGURATION_ERROR.

Преобразует позу из текущей системы координат в заданную исходную систему.

Использует буфер TF2 для выполнения преобразования из input.header.frame_id в source_frame. Проверяет, находится ли результирующая точка в пределах границ, установленных для source_frame.

Parameters
inputВходная поза (PoseStamped) в некоторой системе координат.
outputВыходная поза (PoseStamped), в которую будет записан результат преобразования в системе source_frame.
source_frameИдентификатор системы координат (TF frame ID), в которую нужно преобразовать.
Returns
Статус операции: SUCCESS, OUT_OF_BOUNDS, TRANSFORM_NOT_FOUND, INVALID_INPUT или CONFIGURATION_ERROR.

◆ removeBounds()

void coordinate_transformer::CoordinateTransformer::removeBounds ( const std::string &  frame_id)

Удаляет ранее установленные границы для указанной системы координат.

Если для frame_id были установлены границы, они удаляются из внутреннего хранилища. Соответствующие ROS-параметры не удаляются, но и не используются после вызова этого метода.

Parameters
frame_idИдентификатор системы координат (TF frame ID), для которой удаляются границы.

Устанавливает или обновляет границы для заданной системы координат.

Сохраняет переданные минимальные и максимальные координаты для указанного frame_id. Также объявляет и устанавливает соответствующие ROS-параметры для динамического управления. Если min > max по любой оси, границы не устанавливаются и выводится ошибка.

Parameters
frame_idИдентификатор системы координат (TF frame ID).
minСтруктура geometry_msgs::msg::Point, содержащая минимальные координаты (x, y, z).
maxСтруктура geometry_msgs::msg::Point, содержащая максимальные координаты (x, y, z).
