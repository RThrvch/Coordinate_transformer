 Библиотека C++ и node для преобразования координат с проверкой границ для ROS 2.\hypertarget{index_autotoc_md1}{}\doxysection{\texorpdfstring{Обзор}{Обзор}}\label{index_autotoc_md1}
Этот пакет предоставляет функциональность для управления и запроса преобразований координат (tf) между различными системами координат (фреймами), в первую очередь предназначенную для использования в среде ROS 2. Он позволяет определять статические преобразования и проверять, попадают ли определенные точки или позы в предопределенные границы для заданных систем координат.\hypertarget{index_autotoc_md2}{}\doxysection{\texorpdfstring{Зависимости}{Зависимости}}\label{index_autotoc_md2}

Этот раздел определяет именованные границы, связанные с конкретными системами координат. Каждая граница определяется своими минимальной и максимальной угловыми точками.


\begin{DoxyItemize}
\item {\ttfamily \texorpdfstring{$<$}{<}frame\+\_\+name\texorpdfstring{$>$}{>}}\+: (string) Имя системы координат, к которой применяется граница (например, {\ttfamily map}, {\ttfamily robot\+\_\+workspace}). Должно совпадать с системой координат, определенной в вашей системе (либо через раздел {\ttfamily transforms}, либо иным образом).
\begin{DoxyItemize}
\item {\ttfamily min}\+: (object) Минимальный угол ограничивающего параллелепипеда, выровненного по осям.

\item {\ttfamily y}\+: (double) Минимальная координата y.
\item {\ttfamily z}\+: (double) Минимальная координата z.
\end{DoxyItemize}
\item {\ttfamily max}\+: (object) Максимальный угол ограничивающего параллелепипеда, выровненного по осям.
\begin{DoxyItemize}
